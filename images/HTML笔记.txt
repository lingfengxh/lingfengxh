超链接标签
<a href="跳转目标" target="目标窗口的弹出方式(_self为默认值，_blank为新窗口打开的方式)">文本或图像</a>

链接分类
1.外部链接：例如<a href="http://www.baidu.com">百度</a>
2.内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可，例如<a href="index.html">首页</a>
3.空链接：如果当时没有确定链接目标时，<a href="#">首页</a>
4.下载链接：如果href里面地址是一个文件(.exe)或压缩包，会下载这个文件， 例如<a href="img.zip">下载文件</a>
5.网页元素链接：在网页中的各种网页元素，如文本、图像、表格、音频、视频等都可以添加超链接， 
例如<a href="http://www.baidu.com"><img src="img.jpg"/></a>
6.锚点链接：我们点击链接，可以快速定位到页面中的某个位置。
	*(点击的位置)在链接文本的href属性中，设置属性值为#名字的形式，如<a href="#two">第2集</a>
	*(到达的位置)找到目标位置标签，里面添加一个id属性=刚才的名字，如<h3 id="two">第2集介绍</h3>

合并单元格
跨行合并：rowspan="合并单元格的个数"
跨列合并：colspan="合并单元格的个数"

合并单元格三部曲：
1.先确定是跨行还是跨列合并。
2.找到目标单元格，写上合并方式=合并单元格数量。比如：<td colspan="2"></td>。
3.删除多余的单元格。

无序列表
<ul>标签用于定义无序列表，列表以点来显示。
<li>标签用来定义列表项。
<ul>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ...
</ul>
1.无序列表的各个列表项之间没有顺序级别之分，是并列的。
2.<ul></ul>中只能嵌套<li></li>，直接在<ul></ul>标签中输入其他标签或者文字的做法是不被允许的。
3.<li>与</li>之间相当于一个容器，可以容纳所有元素。
4.无需列表会带有自己的样式属性，在实际使用时，我们会使用CSS来设置。

有序列表
<ol>标签用于定义有序列表，列表顺序以数字来显示。
<li>标签用来定义列表项。
<li>
<ol>
   <li>列表项1</li>
   <li>列表项2</li>
   <li>列表项3</li>
   ...
</ol>
1.<ol></ol>中只能嵌套<li></li>，直接在<ol></ol>标签中输入其他标签或者文字的做法是不被允许的。
2.<li>与</li>之间相当于一个容器，可以容纳所有元素。
3.有序列表会带有自己样式属性，在实际使用时，我们会使用CSS来设置。

自定义列表
<dl>标签用于定义描述列表，该标签会与<dt>(定义项目/名字)和<dd>(描述每一个项目/名字)一起使用。
<dl>
   <dt>名词1</dt>
   <dd>名词1解释1</dd>
   <dd>名词2解释2</dd>
</dl>
1.<dl></dl>里面只能包含<dt>和<dd>。
2.<dt>和<dd>个数没有限制，经常是一个<dt>对应多个<dd>。
3.<dt>、<dd>里面可以包含任何标签。

列表总结
标签名		定义		说明
<ul></ul>	无序列表		里面只能包含li，没有顺序，使用较多。li里面可以包含任何标签
<ol></ol>	有序列表		里面只能包含li，有顺序，使用相对较少。li里面可以包含任何标签
<dl></dl>	自定义列表	里面只能包含dt和dd。dt和dd里面可以放任何标签

表单域
<form>标签会把它范围内的表单元素信息提交给服务器
<form action="url地址" method="提交方式" name="表单域名称">
各种表单元素控件——如<input>/<select>/<textarea>
</form>
常用属性：
属性	属性值	作用
action	url地址	用于指定接受并处理表单数据的服务器程序的url地址
method	get/post	用于设置表单数据的提交方式，其取值为get或post	
name	名称	用于指定表单的名称，以区分同一个页面中的多个表单域

表单标签
<input>标签控件用于收集用户信息
<input type="属性值" name="属性值" value="属性值" checked="属性值" maxlength="属性值">
属性值		描述——type属性
text		定义单行的输入字段，用户可在其中输入文本。默认宽度为20个字符。
password		定义密码字段，该字段中的字符被掩码
radio		定义单选按钮
checkbox		定义复选框
submit		定义提交按钮。提交按钮会把表单数据发送到服务器——可通过value修改提交文字，转到form的url地址
reset		定义重置按钮。重置按钮会清除表单中的所有数据——可通过value修改重置文字
button		定义可点击按钮
file		定义输入字段和"浏览"按钮，供文件上传
hidden		定义隐藏的输入字段
image		定义图像形式的提交按钮

<input>表单元素
属性		属性值		描述
name		由用户自定义	定义input元素的名称
value		由用户自定义	规定input元素的值
checked		checked		规定此input元素首次加载时应当被选中
maxlength	正整数		规定输入字段中的字符的最大长度
1.name和value是每个表单元素都有的属性值，主要给后台人员使用
2.name表单元素的名字，要求单选按钮和复选按钮都有相同的name值
3.checked属性主要针对于单选按钮和复选框，主要作用一打开页面，就可以默认选中某个表单元素
4.maxlength是用户可以在表单元素输入的最大字符数，一般较少使用

<label>标签用于绑定一个表单元素，当点击<label>标签内的文本时，浏览器就会自动将光标转到或者选择对应的表单元素上，用来增加用户体验。
语法：
<label for="t1">男</label>
<input type="radio" name="sex" id="t1"/>
核心：<label>标签的for属性应当与相关元素的id属性相同。

<select>表单元素

	<option checked="checked">选项3</option>	//选中选项3
